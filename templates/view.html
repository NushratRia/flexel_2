💡 Tips

Interaction Modes

🖱 WIMP Mode

The traditional mouse-and-keyboard system for clicking, typing, and navigating in the sheet.

🎙 Voice Mode

Control the sheet using your voice:

  • Click the 🎙 button.
  • Press Speak and say your command (e.g., “sort column A”).
  • The system will execute the task hands-free.

🖐 Gesture Mode

Perform tasks with hand gestures:

  • Selecting: Pinch fingers together on the target cell.
  • Scrolling: Swipe up/down.
  • Undo: Long swipe left with an open palm.
  • Redo: Long swipe right with an open palm.
  • Delete: Select then flick downward.
  • Merge: Select with two hands and collide them.
  • Zoom In/Out: Standard pinch-to-zoom gesture using all fingers.
  • Copy: Pinch to select data, then close the other fist.
  • Paste: Open fist.

⚡ Combination Mode

Use all the above interaction methods together. For example:

  • Point to data with a gesture and say a voice command like “sort this.”
  • Switch between WIMP, Voice, and Gesture seamlessly in the same workflow.
